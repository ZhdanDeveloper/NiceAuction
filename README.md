## NiceAuction - simple WEB API for online auction

1. UI : Swagger
2. Framework : ASP.NET Core
3. Architecture : 3-Tier
4. ORM : EF Core
5. DB : MS SQL
6. Authentication : Microsoft Identity + JWT
7. Roles : User, Admin [Changes in DBMS]
